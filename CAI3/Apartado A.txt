ALG1 -> 7.zip que utiliza el AES-256

ALG2 -> VeraCrypt -> 5 nuevas cascadas de algoritmos de cifrado: 
Camellia-Kuznyechik, Camellia-Serpent, Kuznyechik-AES, 
Kuznyechik-Serpent-Camellia y Kuznyechik-Twofish. Estos agregan más opciones 
al soporte de cifrado múltiple de VeraCrypt.
Camellia es un cifrado de bloque de clave simétrico con un tamaño de 
bloque de 128 bits y tamaños de clave de 128, 192 y 256 bits.
EL volumen de VeraCrypt está completamente encriptado (incluyendo nombres de 
archivos,tablas de asignación, espacio libre, etc.) y se comporta como un disco 
real. Puede guardar (o copiar, mover, etc.) archivos en este disco virtual y se 
cifrarán sobre la marcha a medida que se escriben.

ALG3 -> AxCrypt : Permite codificar, comprimir, decodificar, ver y editar 
archivos mediante el uso del algoritmo AES-128 y SHA-1. Los archivos codificados
 poseen la extensión *.axx y se muestran con un icono diferente.

Para los 3 algoritmos vamos a hacer la prueba de cifrar/descifrar a:
-Un pdf para tener una similitud con los documentos que pueda tener el hospital.
-Una fotografía que simule una radiografía.
-Un fichero txt, que pueda ser utilizado para notas o recetas.

Tamaños:
-PDF: 264 KB
-Fotografía: 24,7 KB
-Fichero.txt: 5,77 KB
